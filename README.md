**\# Uni Notes Website**

**Introduction**

**PRO-NOTES** is a responsive educational website designed to provide students with easy access to subject notes and study materials. It offers a simple, organized, and user-friendly interface built using HTML, CSS, and JavaScript.

**Purpose**

\- Handwritten notes availability at one click

\- Downloadable for offline use

\- Easy exam prep

**Features**

\- Responsive design (mobile & desktop)

\- Login page with roll number validation

\- Downloadable PDF notes

\- Contact form connected to Google Sheets

\- Card-based layout with hover effects

\- 11 page organized content

**Technologies Used**

\- HTML5

\- CSS3 (Flexbox, Animations, Media Queries)

\- JavaScript (Login validation, Navbar)

\- Google Forms (for contact responses)

**Usage**

\- Enter your roll number on the login page

\- Navigate pages from the homepage

\- Navigate required subject from Subjects page

\- Click on PDF buttons to view or download notes

\- Use the contact form to send queries

\- Get information about the platform from the about

page
